# BREACHPOINT

A static production build of **BREACHPOINT** — an original, browser-based 3D
first-person military shooter built from scratch with **Three.js**. No game engine,
no copyrighted assets: every model is primitive geometry, every texture is generated
on a canvas, and every sound is synthesized live with the WebAudio API.

**▶ Play:** open `index.html` (served at `/breachpoint/` on the site).

Features: tight FPS movement (sprint, slide, lean, vault), 5 weapons + grenades,
A\* pathfinding AI with 5 enemy types and a boss, Survival (waves + Armory) and
Operation (capture sectors) modes, loadouts & perks, killstreak airstrike, bloom
post-processing, procedural audio, and a full HUD/menu suite.

This folder is the compiled output (`vite build`). Part of the
[dangerously-skip-permissions](https://github.com/Bummelnderboris/dangerously-skip-permissions)
collective.
